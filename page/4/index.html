#6. ZigZag Conversion

The string “PAYPALISHIRING” is written in a zigzag pattern on a given number of rows like this: (you may want to display this pattern in a fixed font for better legibility)

P A H N
A P L S I I G
Y I R
And then read line by line: “PAHNAPLSIIGYIR”
Write the code that will take a string and make this conversion given a number of rows:

string convert(string text, int nRows);
convert(“PAYPALISHIRING”, 3) should return “PAHNAPLSIIGYIR”.

Solution:

Retrival the elements in the CharArray, use StringBuffer[] sb as rows, and mark the row number to every element. After while loop, the elements are store as row level in sb, then append all the elements in sb into sb[0], the sb[0] will be the result.

#125. Valid Palindrome

Given a string, determine if it is a palindrome, considering only alphanumeric characters and ignoring cases.

For example,
“A man, a plan, a canal: Panama” is a palindrome.
“race a car” is not a palindrome.

Note:
Have you consider that the string might be empty? This is a good question to ask during an interview.

For the purpose of this problem, we define empty string as valid palindrome.

Solution:

Use isLetterOrDigit() to get only char or Integer, then use toLowerCase to let all char be the lowercase, then compare the head from begin and the tail from the end.
if there is any char is not the same, return false, else return true.

#303. Range Sum Query - Immutable

Given an integer array nums, find the sum of the elements between indices i and j (i ≤ j), inclusive.

Example:
Given nums = [-2, 0, 3, -5, 2, -1]

sumRange(0, 2) -> 1
sumRange(2, 5) -> -1
sumRange(0, 5) -> -3
Note:
You may assume that the array does not change.
There are many calls to sumRange function.

Solution:

In the constructor, we can caculate every sum from begin to index in the array and add into int[] sums,then if i==0, just return sums[j], else, we use the sums[j] - sums[i-1] to get the sum between i and j.

add - Add the number to an internal data structure.
find - Find if there exists any pair of numbers which sum is equal to the value.

For example,
add(1); add(3); add(5);
find(4) -> true
find(7) -> false

Solution:

Use hashmap to store the number as key, the value is the count of this number. So in add function, if containskey, increment the value, if not, put the set(number,1) into map.
In find function, from the list to find a pair num1 and num2 which sum equal value, if num1==num2, the count of number must greater than 1, if not, use containsKey to find num2.
